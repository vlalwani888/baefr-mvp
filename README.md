# BaeFr MVP - Developer Setup Guide

## Project Overview
BaeFr is a modern, high-stakes dating application prototype. This repository contains the frontend mobile client built using React Native and the Expo framework (SDK 54).

## Current Directory Structure
* `BaeFr/`
* `├── .expo/` (Expo managed configuration)
* `├── assets/` (Local images, icons, and fonts)
* `├── App.js` (Main application entry point - Splash & Login UI)
* `├── app.json` (Expo configuration file for packaging)
* `├── package.json` (Project dependencies and scripts)
* `└── README.md` (Project documentation)

## Installation & Setup Instructions

Follow these precise steps to pull the code and run the application on your local machine.

### 1. Clone the Repository
Run this in your terminal:
> git clone <YOUR_GITHUB_REPO_URL_HERE>
> cd baefr-mvp

### 2. Install Dependencies
Run this in your terminal:
> npm install

### 3. Start the Development Server
Run this in your terminal:
> npx expo start

### 4. Run on a Physical Device
1. Download the **Expo Go** application from the Apple App Store or Google Play Store.
2. Ensure your mobile device is connected to the same Wi-Fi network as your computer.
3. Scan the QR code displayed in your terminal to instantly launch the application.
